Rainfall Time Slider

About this Project

For this solution, climate precipitation data was downloaded from Climate Data Canada for maximum 1-day total precipitation amounts over a 100-year period starting in 2020 in a tiling format (Climate Data Canada 2020).

These were imported into ArcGIS Pro and a series of geoprocessing tools were run to put them into a readable spatial format. Once the points were loaded from the tile layers, a kriging interpolation was run at 10-year intervals.

Once these raster surfaces were generated, they were compiled into a raster mosaic dataset.
